/*
    COUNTING SORT ANALYSIS

    Time Complexity:
    Best Case:    O(n + k)
    Average Case: O(n + k)
    Worst Case:   O(n + k)

    Where:
    n = number of elements in array
    k = maximum value in array

    Explanation:
    - Finding maximum element takes O(n).
    - Creating and initializing counting array takes O(k).
    - Counting occurrences of each element takes O(n).
    - Rebuilding the sorted array takes O(n + k).
    - Total complexity = O(n + k).


    Space Complexity:
    O(k)

    Explanation:
    - Extra counting array of size (maximum element + 1) is created.
    - No recursion or additional data structures are used.


    Algorithm Type:
    Non-comparison based sorting algorithm.

    Stable:
    No
    (This implementation directly places values and does not preserve
    the relative order of equal elements.)

    In-place:
    No
    (Uses an additional counting array.)
*/
